破解 Win11 开机密码｜多途径解决忘记密码的困境

Eilo2025-06-192025-06-19

如果你忘记了Windows 11电脑的开机密码，或者需要绕过密码进入系统，可以尝试以下几种方法。

方法一：使用Microsoft账户重置密码（最推荐）

如果你的Windows 11账户是与Microsoft账户关联的，这是最简单和官方的重置密码方法。

  1. 访问Microsoft账户重置页面： 在另一台可以上网的设备上，打开浏览器访问 account.live.com/password/reset。
  2. 输入你的Microsoft账户： 按照提示输入与你Windows 11电脑关联的Microsoft邮箱、电话号码或Skype名称，然后点击“下一步”。
  3. 验证身份： Microsoft会提供几种验证你身份的方式，例如通过发送验证码到备用邮箱或手机。选择一种方式并完成验证。
  4. 设置新密码： 验证成功后，你就可以设置一个新的密码。
  5. 使用新密码登录Windows 11： 设置好新密码后，回到你的Windows 11电脑，使用新密码登录即可。

方法二：使用安全模式和内置管理员账户（如果已启用）

此方法适用于你忘记了用户密码，但内置管理员账户未被禁用，或者你可以在安全模式下访问它。默认情况下，内置管理员账户是禁用的。

  1. 进入Windows 恢复环境（WinRE）：
    • 在开机时，当看到Windows Logo出现时，长按电源按钮强制关机。重复此操作两到三次，直到电脑进入“恢复”界面。
    • 或者，在登录界面，按住Shift键不放，点击电源图标，然后选择“重启”。
  2. 选择疑难解答： 在“选择一个选项”屏幕上，点击“疑难解答”。
  3. 选择高级选项： 点击“高级选项”。
  4. 选择启动设置： 点击“启动设置”，然后点击“重启”。
  5. 进入安全模式： 重启后，你会看到一个选项列表。按键盘上的4或F4键选择“启用安全模式”。
  6. 使用管理员账户登录： 在安全模式下，尝试使用内置的“Administrator”账户登录（如果它有密码，你需要知道密码）。如果成功登录，你可以修改其他用户账户的密码。

方法三：使用密码重置盘（如果之前创建过）

如果你之前在Windows 11电脑上创建了密码重置盘（USB闪存驱动器），这是最方便的本地重置密码方式。

  1. 插入密码重置盘： 将你之前创建的密码重置盘插入锁定的电脑。
  2. 点击“重置密码”： 在Windows 11登录界面，当你输入错误密码后，通常会出现“重置密码”链接。点击它。
  3. 按照向导操作： 按照屏幕上的提示，选择你的密码重置盘，然后输入新密码和密码提示。
  4. 完成密码重置： 完成后，你就可以使用新密码登录了。

方法四：使用安装介质重置密码（复杂，但有效）

如果以上方法都不适用，你可以使用命令提示符来重置密码。这个方法相对比较复杂一些。

  1. 进入Windows 恢复环境（WinRE）：

    • 在开机时，当看到Windows Logo出现时，长按电源按钮强制关机。重复此操作两到三次，直到电脑进入“恢复”界面。
    • 或者，在登录界面，按住Shift键不放，点击电源图标，然后选择“重启”。

  2. 进入命令提示符：

    • 在恢复界面，选择“疑难解答” > “高级选项” > “命令提示符”。

  3. 替换实用工具管理器为命令提示符：

    • 在命令提示符中，输入以下命令（假设你的Windows安装在C:盘，如果不是，请替换为正确的驱动器号）：
    • 问你**覆盖 C:\Windows\system32\utilman.exe 吗？(Yes/No/All):**的时候输入 Yes。
    • 这些命令会将“轻松使用”按钮（utilman.exe）替换为命令提示符（cmd.exe）。

  4. 重启电脑并访问命令提示符： 返回恢复页面，选择“继续”。在登录界面，点击右下角的“轻松使用”图标。此时会打开命令提示符。

  5. 重置密码： 在命令提示符中，输入以下命令来查看账户名称：

    • 例如：Eilo （那“Eilo”就是你的你的账户名）。
      输入以下命令来为这个账户重置密码：
    • 例如：net user Eilo * （将“用户名”替换为你的账户名，一定是小写星号）。
    • 接下来他就允许你重置密码了，你输入一个新的密码即可。
